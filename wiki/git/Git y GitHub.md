# Git y GitHub

Git es una herramienta local de control de versiones. GitHub es el servidor remoto donde se almacenan y comparten esos repositorios.

---

## Ruta de Aprendizaje

| #   | Módulo      | Descripción                                 |
| --- | ----------- | ------------------------------------------- |
| 1   | Fundamentos | Instalación, configuración y primeros pasos |
| 2   | Commits     | Guardar versiones y navegar el historial    |
| 3   | Ramas       | Trabajar en paralelo y fusionar cambios     |
| 4   | GitHub      | Subir código a la nube y colaborar          |
