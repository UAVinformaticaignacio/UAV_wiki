# ROS2

**ROS2** (Robot Operating System 2) es un framework para construir software de robots: middleware de comunicación (DDS) entre nodos, herramientas de desarrollo y un ecosistema de paquetes.

---

## Conceptos básicos

- **Nodo** — un proceso que realiza una tarea (publicar sensores, controlar motores, planificar rutas).
- **Topic** — canal de comunicación donde los nodos publican y se suscriben a mensajes.
- **rclpy** — biblioteca cliente de ROS2 para Python.
